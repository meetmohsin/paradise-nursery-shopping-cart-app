# Paradise Nursery Shopping Cart App

Paradise Nursery is a React shopping cart application built as a plant shop experience. The app opens with a landing page that introduces the company, then lets users explore grouped plant categories, add products to the cart, and manage quantities before checkout.

## Project Features

- Landing page with the Paradise Nursery company name and a Get Started button
- About Us section with company details
- Product listing page grouped by plant categories
- Add to Cart flow with disabled state after an item is added
- Shopping cart with increment, decrement, delete, total quantity, and total amount

## Tech Stack

- React
- Redux Toolkit
- React Redux
- Vite
